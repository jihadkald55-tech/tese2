# نظام إدارة الأدوار والمستخدمين - مداد

## نظرة عامة

تم تحديث نظام مداد ليدعم إدارة الأدوار المختلفة بشكل كامل مع نظام تواصل متطور.

## الأدوار المتاحة

### 1. طالب (Student)
- الوصول إلى صفحة البحث الشخصي
- إدارة المصادر والمراجع
- التواصل مع المشرف
- متابعة التقدم
- جدولة المواعيد

### 2. أستاذ (Professor)
- متابعة الطلاب المسجلين تحت إشرافه
- مراجعة الأبحاث
- التواصل مع الطلاب
- إضافة تعليقات وملاحظات
- إدارة الجداول

### 3. مدير (Admin)
- إدارة النظام
- عرض إحصائيات شاملة
- إدارة المستخدمين

## المستخدمين الافتراضيين

للاختبار، يمكنك استخدام الحسابات التالية:

### طالب
- البريد: `student@university.edu`
- كلمة المرور: `123456`
- الاسم: أحمد محمد
- المشرف: د. محمد العلي

### أستاذ
- البريد: `prof@university.edu`
- كلمة المرور: `123456`
- الاسم: د. محمد العلي
- الطلاب: أحمد محمد

### مدير
- البريد: `admin@university.edu`
- كلمة المرور: `123456`
- الاسم: إدارة النظام

## ميزات نظام التواصل

### البحث عن مستخدمين
- للطالب: يمكن البحث عن الأساتذة فقط
- للأستاذ: يمكن البحث عن الطلاب فقط
- بحث بالاسم أو البريد الإلكتروني

### إنشاء محادثة جديدة
1. اضغط على زر "بحث عن مشرف" (للطالب) أو "بحث عن طالب" (للأستاذ)
2. ابحث عن المستخدم المطلوب
3. اختر المستخدم لبدء المحادثة

### إدارة المحادثات
- عرض جميع المحادثات
- تثبيت المحادثات المهمة
- كتم الإشعारات
- أرشفة المحادثات
- حذف المحادثات

## التقنيات المستخدمة

- **Context API**: لإدارة حالة المستخدم
- **LocalStorage**: لحفظ البيانات محلياً
- **TypeScript**: للتحقق من الأنواع
- **Framer Motion**: للحركات والانتقالات

## كيفية الاستخدام

### 1. تسجيل الدخول
```
افتح الصفحة الرئيسية
اختر نوع المستخدم (طالب/أستاذ/مدير)
أدخل البريد الإلكتروني وكلمة المرور
اضغط على "تسجيل الدخول"
```

### 2. إنشاء حساب جديد
```
اذهب إلى صفحة التسجيل
املأ البيانات المطلوبة
اختر نوع الحساب
اضغط على "إنشاء الحساب"
```

### 3. التواصل مع مستخدمين آخرين
```
اذهب إلى صفحة المحادثات
اضغط على "بحث عن مستخدم"
ابحث عن الشخص المطلوب
اختره لبدء المحادثة
```

## البيانات المحفوظة

يتم حفظ البيانات التالية في LocalStorage:

- `currentUser`: بيانات المستخدم الحالي
- `users`: قائمة جميع المستخدمين
- `chatConversations`: المحادثات
- `researchSources`: المصادر والمراجع
